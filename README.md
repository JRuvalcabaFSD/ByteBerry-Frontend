# ByteBerry Frontend

Sistema de Gestión de Gastos - Frontend Service

## Stack Tecnológico

- React 18
- TypeScript 5.6
- Vite 6
- pnpm 10.17.1
- Clean Architecture

## Desarrollo Local
```bash
# Instalar dependencias
pnpm install

# Levantar servidor de desarrollo (puerto 5173)
pnpm dev

# Lint
pnpm lint

# Format
pnpm format

# Build para producción
pnpm build

# Preview build (puerto 4003)
pnpm preview
